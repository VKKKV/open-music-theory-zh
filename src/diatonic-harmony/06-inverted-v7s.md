# 概述

西方古典音乐中的乐句开头通常以主和声的延续为特征，以确立主调。"延续"意味着该和声的影响持续超过一个单独的和弦。比如说，你买了一份想要慢慢享用的冰淇淋。一种方法是小口小口地吃以延长吃的时间。虽然这种方法可行，但到最后你可能吃的是冰淇淋汤。另一种方法是吃一些，放进冰箱，做些别的事情，然后再回来继续吃。你可能会说"我吃了一整天冰淇淋"，尽管你并不是字面上每秒都在吃冰淇淋（尽管你可能想这样）。

音乐中发生了类似的事情。我们可以在乐句开头通过保持或重复和弦来延续主功能的重要性（就像小口吃冰淇淋），如例1所示，但更有趣和更有收获的是在主和弦实例之间使用其他和弦（就像把冰淇淋放进冰箱稍后再回来）。

例1.通过保持或重复和弦进行延续。

例2展示了在乐句开头延续主功能最常见方式的一个实例：在主音三和弦之间使用转位 V7。主功能的影响力更强，因为 (1) 它在比 V7 更强的拍子或超级拍子上，(2) 它至少出现一次原位，而 V7 处于较弱的转位。

例2.克拉拉·舒曼（Clara Schumann），钢琴三重奏，第三乐章，第1–2小节中的主功能延续（0:00-0:08）。

# 书写主功能延续

本章涵盖的主功能延续是西方古典音乐中最常见的类型，它们都具有以下特征：

- 它们长为三个和弦
- 第一个和最后一个和弦是 I 或 I6
- 中间的和弦是 V6 或转位 V7
- V6 或转位 V7 使用我们在用终止六四和弦加强结束感中学到的相同原则解决

## 用 V6/5 和 V6 延续

$\mathrm{V}^6_5$ 通常在进行 $\mathrm{I-V^6_5-I}$ 中延续原位 I（例3）。这是因为ti（$\hat{7}$）在低音声部，我们知道ti（$\hat{7}$）必须解决到do（$\hat{1}$）。一如既往，遵循典型的书写程序。

例3.涉及 $\mathit{V^6_5}$ 的主功能延续。

也可以用 V6 而非 $\mathrm{V}^6_5$ 延续主功能，尽管这不太常见（例4）。

例4.涉及 $\mathit{V^6}$ 的主功能延续。

## 用 V4/2 延续

$\mathrm{V}^4_2$ 通常帮助我们从原位 I 移动到第一转位 I，在进行 $\mathrm{I-V^4_2-I^6}$ 中（例5）。这是因为 $\mathrm{V}^4_2$ 的低音是fa（$\hat{4}$），它必须解决到mi（$\hat{3}$），因为fa（$\hat{4}$）是和弦的七音。再次，遵循典型的书写程序。

例5.涉及 $\mathit{V^4_2}$ 的主功能延续。

## 用 V4/3 延续

最常见的，$\mathrm{V}^4_3$ 在进行 $\mathrm{I-V^4_3-I^6}$ 中帮助我们从原位 I 移动到第一转位 I（例6）。它偶尔在进行 $\mathrm{I-V^4_3-I}$ 中延续原位主和弦，但这不太常见（例7）。这些选择是可行的，因为 $\mathrm{V}^4_3$ 的低音re（$\hat{2}$）可以进行到do（$\hat{1}$）或mi（$\hat{3}$）。再次，遵循典型的书写程序。

例6.涉及 $\mathit{V^4_3}$ 的主功能延续。

例7.不太常见的涉及 $\mathit{V^4_3}$ 的主功能延续。

使用 $\mathrm{V}^4_3$ 书写还提供了一个例外，即和弦的七音fa（$\hat{4}$）必须下行解决的规则（例8）。这里，fa上行到sol（$\hat4-\hat5$）。这个例外之所以可行，是因为低音与取线mi–fa–sol（$\hat3-\hat4-\hat5$）的上方声部形成了平行十度。

例8.和弦七音典型解决方式的例外。
