VIII. 二十世纪与二十一世纪技法

用集合论分析（或者不用！）Mark Gotham 与 Megan Lavengood
要点总结

- 要意识到你为什么将某些集合归为一组而不包括其他集合（分节/segmentation）。
- 通过将集合相互关联来建立分析的统一性。
- 记住作曲家在集合论（set theory）被发明之前就已经创作了自由无调性（free atonal）音乐。
- 注意集合论的局限性：它不讨论音乐的任何非音高方面，甚至忽略了音高的许多方面。

使用音级集合分析音乐存在许多问题——有些是许多分析形式普遍存在的，有些则与集合论更为相关。

集合论的灵活性是一把双刃剑。因为任何一组音高都可以构成一个集合（set），分析者可以超越三和弦式和声，基于任何形式的音高组织来分析音乐。但同样地，因为任何一组音高都可以构成一个集合，分析者可以"证明"他们想要证明的任何东西。例如，考虑一个动机分析（motivic analysis），其中动机是一个单一音程。通过在所有可能的地方定位这一音程，分析者可以声称一部作品具有动机统一性。虽然这一音程确实可能贯穿整部作品，但其出现的频率本身不太可能具有意义，因此需要分析者来说明它在给定语境中如何以及为何对他们具有音乐意义。简而言之，分析者应始终意识到其观察的基础、意义和可感知性。

以下是使用集合论的一些潜在陷阱，以及加强你使用集合论所做论证的方法。

# 分节

对数字（通过音级整数）的强调可能使集合论看起来比实际上更"科学"。由于集合论关系是事实性的——例如，一个音级集合（pitch-class set）与另一个集合之间存在移位关系，或者不存在——很容易得出一个不可证伪的分析，而该分析在描述音乐方面可能仍然无效。例如，每一首包含音符的作品都可以用单音"集合"来描述，而所有这些集合都彼此等价。显然，这不是一个有用的分析论断！

确保你的集合论论证有用的最佳方法是意识到你的分节（segmentation）过程：你如何将作品分成音级集合。以下是分析者可能将音符归为一组的一些原因：

- 分组的连续性。将同时发生或紧接着发生的音符归为一组有充分理由。你可以为将相距很远的音符归为一组而排除其间许多其他音符进行论证，但这需要更多的解释。
- 共享的节奏轮廓。根据休止符的分隔、共享的节奏动机或任何其他节奏联系来分节集合是很常见的。
- 共享的节拍位置
- 共享的音色和织体
- 共享的演奏法
- 共享的音区（高音区、低音区）

Joseph Straus（2016, 70）精辟地总结了分节的重要性：

> 在你所有的音乐分节中，努力在富有想象力的探索和音乐常识之间取得平衡。一方面，不要局限于显而易见的分组（尽管这通常是一个好的起点）。有趣的关系可能在第一遍、第二遍甚至第三遍时并不明显，你需要在调查中保持彻底和执着。另一方面，你必须保持在能够被有意义地听到的范围之内。你不能仅仅因为某些音符构成了你感兴趣的集合，就以某种随意的方式把它们挑出来。

# 集合之间的关系

仅仅识别集合本身并不具有分析上的趣味性。以下是将集合相互关联并在它们之间建立统一感的一些方法。

- 找到同一音级集合的多个实例，然后通过移位和倒影将它们相互关联。你可以计算从一个集合到下一个集合的移位/倒影关系，或从第一个集合到所有其他集合的关系，或任何其他配置。你的关系应该都涉及乐谱中实际出现的音符——不要费力去将集合与对应于原型的标准序进行关联，例如，除非该标准序确实出现在作品中。
- 看看不同的集合是否属于同一个超集（superset）。该超集可能是一个熟悉的调式或音级集合。
- 通过对称于一个倒影轴线或强调某一音级中心，将集合间的音符对相互关联。

# 实践先于理论

从更广泛的角度来看，即使不是大多数，也有许多音乐理论是回顾性的：处理某个曲目的术语和技术是在事后产生的；它们的存在是为了理解已经存在的音乐。这并非总是如此：许多作曲家-理论家发展了关于音乐如何组织的想法，然后将这些想法付诸实践。序列主义（serialism）就是一个重要的例子。但集合论是一种回顾性理论，在它所针对的曲目创作数十年后才由艾伦·福特（Allen Forte）推广开来。因此，我们应该注意将集合论与这些作曲家的创作行为和意图区分开来。例如，阿诺德·勋伯格（Arnold Schoenberg, 1950）认为自由无调性音乐的结构是不可能的，因此他随后发展了序列技术。

仅仅因为一个理论是事后发展出来的，并不意味着该理论或多或少是有效的（作曲家在历史上至少在混淆其方法方面与澄清方法方面一样活跃！），但值得牢记这一点。

# 集合论不会告诉你的事情

请务必考虑集合论在任何程度上都无法涉及的方面：

- 历史和语境
- 任何非音高参数（尽管这取决于分析者如何处理诸如分节之类的问题）
- 音高参数，如声部进行、间距或顺序（同样，取决于分析方法）

这个列表包括许多对听众和演奏者来说都很重要的音乐方面。你可能需要注意通过其他方式在分析中涉及这些问题，因为集合论分析常常会忽略它们。
