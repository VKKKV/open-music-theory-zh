VIII. 二十世纪与二十一世纪技法

音级集合（Collections）Mark Gotham; Megan Lavengood; Brian Moseley; 与 Kris Shaffer
要点总结

本章介绍了若干在二十世纪吸引众多作曲家的音级集合：

- 自然音级集合（diatonic collection），有别于大/小调音阶或自然音调式
- 五声音阶集合（pentatonic collection）：一个五音集合，对应钢琴的黑键；也可以通过叠加五个纯五度或通过音程模式 2-2-3-2-3 生成
- 全音音阶集合（whole-tone collection）：一个六音集合，完全由全音间隔的音符组成
- 八声音阶集合（octatonic collection）：一个八音集合，由全音和半音交替构成
- 六声音阶集合（hexatonic collection）：一个六音集合，由音程模式 1-3-1-3-1-3 生成
- 泛音音阶集合（acoustic collection）：一个七音集合，类似于混合利底亚调式但带有 $\uparrow\hat{4}$；大致对应于泛音列（harmonic series）的最低分音

本章还介绍了其他音级集合：

- 奥利维耶·梅西安（Olivier Messiaen）的"有限移位调式"（modes of limited transposition）：一组无法以12种不同方式移位的音阶
- 用重复的音程模式生成音级集合的"距离模型"（distance model）

除了大小调和自然音调式的使用之外，还有四种新的音级集合在二十世纪占有特殊地位：五声音阶、全音音阶、八声音阶和泛音音阶。本章讨论这些集合以及关于调式特性和音乐外意义的一些重要问题。

# 自然音级集合与全自然音主义

你可能已经多次遇到自然音级集合的概念，特别是在关于流行音乐调式和二十/二十一世纪音乐的章节中，或者当你第一次学习大调和小调时。自然音级集合是许多西方音乐的基础。

在二十世纪和二十一世纪，作曲家有时使用自然音级集合，但不试图使某个特定音高听起来像音高中心。这样的例子既不是调性的，也不是调式的；相反，它们被认为是全自然音的（pandiatonic）。伊戈尔·斯特拉文斯基（Igor Stravinsky）经常写作全自然音段落；许多可以在他的芭蕾舞剧《彼得鲁什卡》（Petrushka）的开头部分听到。

# 五声音阶集合

五声音阶集合（pentatonic collection）在全球音乐中非常普遍。它是一个由音程模式大二度-大二度-小三度-大二度-小三度构成的集合。五声音阶集合可以用多种不同方式参照自然音调式来描述（例 1）。

- 使用大调音阶的音级，五声音阶可以用 $\hat{1}\text{, }\hat{2}\text{, }\hat{3}\text{, }\hat{5}\text{, and }\hat{6}$ 构成。
- 如果你把自然音级集合想象成一组纯五度叠加（F, C, G, D, A, E, B），那么五声音阶就是同一集合的较窄形式：C, G, D, A, E。
- 最后，如果你想象钢琴，注意 C, D, E, F, G, A, B, C（白键）的补集是 G♭, A♭, B♭, D♭, E♭（黑键）。黑键集合同样就是五声音阶。

例 1. (a-c) 生成五声音阶集合的三种方式，以及 (d) 五声音阶集合的一种转位。

五声音阶集合从自然音中去除的是产生半音的两个音符（如果你以大调音级来思考，就是 $\hat{4}$ 和 $\hat{7}$）。半音的缺失可以说使五声音阶更容易被转位并被视为一个集合，而不会特别强调某个音符作为主音。集合中的任何成员都容易充当音高中心。例如，例 1 中 (a) 的集合有五种不同的调式，通过将集合转位使每个音级成为主音来形成：C 五声音阶（C, D, E, G, A），D 五声音阶（D, E, G, A, C），E 五声音阶（E, G, A, C, D），依此类推。例 1 的 (d) 给出了五声音阶集合的一个特别常见的转位，被称为小调五声音阶（minor pentatonic）。

五声音阶特别常见于作为旋律创作的基础。一个例子可以在陈怡（Chen Yi）打击乐协奏曲第一乐章中听到，在弦乐进入时的 2:13 处。注意五声音阶的旋律有时由铜管的无调性和声伴奏。五声音阶的旋律不需要在五声音阶集合内和声化，事实上通常不是；毕竟，在五声音阶集合中只有两个可能的三和弦（例如，在集合 C-D-E-G-A 中有 C 大三和弦和 A 小三和弦）。

# 全音音阶集合

全音音阶集合（whole-tone collection）正如其名：一个完全由六个全音组成的音阶。类似于五声音阶，全音音阶在转位上是模糊的，因为只有一种音程大小。作曲家经常利用这种模糊性，使用全音音阶在听者中产生不安的感觉（在电影和电视中，一个著名的套路是使用全音音阶来伴奏梦境场景）。

只有两种独特的全音音阶集合：一种包含偶数音级 [0, 2, 4, 6, 8, 10]，另一种包含奇数音级 [1, 3, 5, 7, 9, 11]。如果你需要在分析中区分这两种集合，可以使用缩写 WT0 和 WT1："WT"代表"全音"（whole tone），下标数字表示音高 C（0）或 C♯（1）（例 2）。

# 八声音阶集合

八声音阶集合（octatonic collection）由全音和半音的交替构成，总共产生八个不同的音高（例 3）。爵士音乐家称之为减音阶（diminished scale），因为它与减七和弦（fully diminished seventh chord）完美契合。（当然，也可以有其他包含八个不同音高的音阶，但这个被称为八声音阶。）


<iframe src="https://musescore.com/user/32728834/scores/5990299/embed" width="100%" height="240" frameborder="0" allowfullscreen allow="autoplay"></iframe>

例 3. 只有三种八声音阶集合。

该集合的音程内容非常均匀，这种音程一致性导致了其最有趣的特性之一。如例 3 所示，只有三种可能的八声音阶集合。当我们把上面第一个集合（OCT0,1）移位 3 时——将集合中的每个整数加 3——<0, 1, 3, 4, 6, 7, 9, 10> 变成 <3, 4, 6, 7, 9, 10, 0, 1>，这与第一个集合相同，只是从不同的音高开始。如果你需要在分析中区分这些集合，可以使用缩写 OCT0,1、OCT1,2 和 OCT2,3："OCT"是"八声音阶"的缩写，下标数字代表在该音阶中构成半音的音高（分别为 C,C♯; C♯,D; 和 D,E♭）。

Joan Tower 经常使用八声音阶集合，在她的作品《银梯》（Silver Ladders）的开头尤其明显。

与这里讨论的其他集合不同，八声音阶集合在二十世纪之前就以一定频率出现，尤其是在俄罗斯。[1]八声音阶集合可以产生几种熟悉的三和弦式和声，如例 4 所示：八个大/小三和弦（以及八个减三和弦，未显示），以及除大七和弦外每种七和弦各四个。然而，没有以五度根音运动（root motion）关联的和弦，因此不可能有主/属运动。相反，有大量以三度进行的根音运动。弗雷德里克·肖邦（Frédéric Chopin）在他的 G 小调叙事曲中在 B♭7 和弦上使用了八声音阶（例 5）。由于所有这些三和弦式和声，它是一种多功能的调式，既可以暗示调性关联，又可以在传统上不被认为密切相关的不同调性之间自由运动。

# 六声音阶集合

六声音阶集合（hexatonic collection）是一个六音集合，由小二度和小三度交替构成。"六声音阶"一词指其六个音符，虽然可能有其他包含六个音符的音阶（例如蓝调音阶/blues scale），但"六声音阶集合"这一名称始终特指这组音符。与八声音阶集合类似，六声音阶集合只能移位四次就会回到相同的音符组（例 6），它们同样以最低半音命名（例如 HEX0,1 是包含 C-C♯ 的六声音阶集合）。

# 泛音音阶集合

泛音音阶集合（acoustic collection）基于泛音列（overtone series）的最低音程。这很重要，因为长期以来这些最低音程与音乐协和性的概念之间存在关联。结果是一种类似于大调音阶但带有 $\uparrow\hat{4}$ 和 $\downarrow\hat{7}$ 的调式（例 8）。

# 组织音高的新方式

二十世纪和二十一世纪的许多作曲家寻找了生成其音乐所用音级集合的新方法。

## 距离模型调式

距离模型调式（distance model modes）涉及一组更受限的有限移位调式。这些调式由两种音程的交替构成，例如：

- 1:2 = 半音和全音交替（再次是八声音阶）
- 1:3 = 半音和小三度，有时称为六声音阶/集合，或用音级集合（pitch-class set）术语来说，"魔力"六音集合（"magic" hexachord）
- 1:5 = 半音和纯四度

这种组织方式与贝拉·巴托克（Béla Bartók）密切相关，我们将在下一章中看到。

### 其他调式

这甚至对于二十世纪古典音乐来说也只是冰山一角。其他主要探索领域包括：

- 微分音调式（Microtonal modes），深受利盖蒂（Ligeti）和频谱乐派的喜爱，专注于替代调律体系，避免对平均律十二音的任何假设。
- 合成调式（Synthetic modes），通过对自然音进行改变而得来，如巴托克《小宇宙》（Mikrokosmos）某些乐章中的非常规调号。
- 一次性案例如威尔第《圣母颂》（Ave Maria）中的"谜语音阶"（scala enigmatica）（在其他任何情况下几乎不被用作音阶/调式）。

# 关于音级集合的重要考量

为什么其中一些调式在不同语境中不断出现？同样，这是一个吸引了大量理论关注的重大问题。以下是一些要点：

- 与自然泛音列的对应。一种假设是，人们喜欢那些重要音高与泛音列低音对齐的调式，从而与传统所称的协和音程（八度、五度等）对齐。显然，泛音音阶集合是这一理念的特别直接的实现。
- 对称性（Symmetry）是二十世纪作曲家的关键关注点。原因之一是希望创造一种新的秩序，不植根（有意双关……）于泛音列和根音低音和声的"从低音向上"的世界。对称性可以是音阶内部的，如有限移位调式的旋转对称，也可以是音阶之间的，如巴托克的《世俗康塔塔》（Cantata Profana）。
- 最大均匀性（Maximal evenness）。一种关于调式构建的重要理论强调音高在空间中的均匀分布。[3]再次想想自然音调式。自然音级集合主要由全音组成，只有两个半音尽可能远离彼此。半音之间的最大间距意味着自然音级集合的音高是最大均匀的（maximally even）。
