要点

# 沥青

音高是具有单独频率的离散音调。

因此，音高的概念并不意味着八度等价。 C4是一个音高，和C3不是同一个音高。

# 音高等级

纵观整个理论，“类”一词的意思是“群体”。因此，音级是一组音高——所有音高都通过八度等效和等和音相关。 您以前可能遇到过这两个概念，即使没有提到名称。

我们的音符字母名称系统意味着八度等效：拼写相同但相距任意数量的八度的音高之间的等效。 C4 与 C3 相同，与 C9 相同，依此类推，因为它们都是 C（示例 1）。

当有人说“有十二个音符”时，他们暗示的是和谐等价。如果您按下钢琴上的同一个琴键来弹奏两个音符，则这两个音符是等音等效的——即使拼写不同。等和谐等价是指 A♭ 和 G♯ “相同”。虽然调性音乐几乎总是区分等音音高 - A♭ asle$(\downarrow\hat{6})$ 导致 G，但 G♯ assi$(\uparrow\hat{5})$ 导致 A - 后调性音乐通常是不同的。[1]因为许多作曲家不再感到受到调性中心的限制，音阶程度和拼写之间的关系并不重要。

总之，音级是由八度和等音等效性相关的音高组。 A♭4、A♭3、G♯2 等都是同一音级的成员。

# 整数表示法

示例 2.
十二个音级表示为钟面。

如果音符是等音等效的，那么七个字母名称的系统不能很好地描述十二个音级。相反，在集合论中，我们使用整数表示法，为每个音级分配一个 0 到 11 之间的数字（示例 2）。所有 C 以及任何与 C 等音等效的音符（例如 B♯）均为音级 0。所有 C♯ 以及与 C♯ 等音等效的任何音符（例如 D♭）均为音级 1，依此类推。总结一下一些最常见的音符名称：

- C（B♯等）
- C♯、D♭
- D（C𝄪等）
- D♯、E♭
- E（F♭等）
- F（E♯等）
- F♯、G♭
- G（F𝄪等）
- G♯、A♭
- A（G𝄪等）
- A♯、B♭
- B（C♭等）
