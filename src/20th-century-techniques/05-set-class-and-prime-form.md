集合类与质数形式
Megan Lavengood 和 Brian Moseley
要点

- 集合类（set class）是一组通过移位或倒影相关联的音级集合。
- 集合类以其质数形式（prime form）命名，即移位至零且最紧凑靠左的版本（与其倒影相比较）。
- 你可以通过数学方法或使用钟面来找到质数形式。
- 所有可能的集合类都汇总在集合类表中，可在维基百科和许多其他网站上查阅。

定义集合类最简单的方式是"一组通过移位或倒影相关联的音级集合"。这起初可能令人困惑，但它只是另一种"类"。正如你在其他章节中学到的，"类"是"组"的另一种名称。回顾一下你已经学过的其他类型的类。

- 音高与音级：音高出现在特定的八度中，我们通常以特定的拼写来理解它。音级（pitch class）是一组通过移位或倒影相关联的音高。
- 音程与音程级：音程具有特定的半音距离，而音程级（interval class）是一组互为倒影或八度相关的音程。

# 简介

音级集合与集合类（音级集合类）是本章的主题。"集合类"的定义可能看起来更令人困惑的原因在于它涉及两种类型的组：类和集合。

- 类是以某种方式相关联的一组事物。
- 集合是不一定以任何特定方式相关联的一组事物。

打个比方，想想生物学中不同生物的分类方式。同一类的植物在生物学上以特定方式相关联：被子植物门（Angiospermae）是一个开花植物的类。但我们可以出于其他原因将植物归为一组：例如某人家前院的植物。那是一组植物，但不是一类植物。

因此，音级集合是分析者出于某种原因决定放在一起的一组音高。音级集合类——这个术语非常冗长，因此理论家们同意将其缩短为后两个词，集合类——是所有通过移位或倒影相关联的音高组的集合。

## 为什么是移位和倒影？

分析大量后调性音乐的一种方式是研究音级集合之间的移位和倒影关系。看下面这个简短的例子：Béla Bartók"主题与倒影"中的两个段落（例1）。比较两个段落，右手的两个集合[10, 0, 2, 3, 5]和[3, 5, 7, 8, 10]通过T5相关联。两个左手集合也以相同方式相关联。再看每个段落内部，右手和左手通过倒影相互关联。在第一个段落中，它们通过I8相关联；在第二个段落中，通过I6相关联。

例1. "主题与倒影"段落之间的Tn和In关系。

为了快速解释为什么这些音符片段听起来相同，我们可以说它们都是同一集合类的成员。

大三和弦和小三和弦可能是一个更有帮助、更熟悉的例子。与音簇、四度和声，甚至增三和弦或减三和弦相比，所有大三和弦和小三和弦听起来都"一样"。原因是所有大三和弦和小三和弦之间都是移位或倒影相关的。

- 相同性质的三和弦是移位相关的。C大三和弦[0, 4, 7]的T2是D大三和弦[2, 6, 9]。A小三和弦[9, 0, 4]的T2是B小三和弦[11, 2, 6]。
- 相反性质的三和弦（大调与小调）是倒影相关的。C大三和弦[0, 4, 7]的I0是F小三和弦[5, 8, 0]。C大三和弦[0, 4, 7]的I2是G小三和弦[7, 10, 2]。

# 质数形式

正如音级集合以其正规形式命名一样，集合类以其质数形式命名：即移位至零且最紧凑靠左的版本（与其倒影相比较）。

注意质数形式只是集合类的一个标签。它没有特殊地位——如果作曲家使用[0, 1, 4]作为音级集合，仅仅因为它与质数形式(014)共享相同的整数，这并不具有特殊意义。

## 数学过程

以下是将音级集合转换为质数形式的过程，以例1中的动机为例。

步骤 | 示例

1. 将音级集合排列为正规顺序。 | [10, 0, 2, 3, 5]
2. 将集合移位使第一个音级为0。 | T2= [0, 2, 4, 5, 7]
3. 将步骤2的结果进行倒影并排列为正规顺序。 | I0= [5, 7, 8, 10, 0]
4. 将步骤3的集合移位使第一个音级为0。 | T7= [0, 2, 3, 5, 7]
5. 比较步骤2和步骤4的集合。最紧凑靠左的集合即为质数形式。将质数形式用括号书写，不含逗号。 | (02357) 是质数形式（例1）。
