## 简介

Open Music Theory (OMT2) 是一本原生在线的开源教育资源，面向本科音乐理论课程。
涵盖传统核心内容（基础、自然音和声、半音和声、曲式、20世纪技法）以及爵士、
流行音乐、节奏、对位法、配器法等扩展单元。本版还包含完整的练习册。
