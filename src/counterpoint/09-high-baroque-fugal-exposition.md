II. 对位法与华丽风格图式

高巴洛克赋格呈示部（High Baroque Fugal Exposition）Mark Gotham
要点总结

- 赋格（fugue）是一种至少自十八世纪以来就很流行的对位体裁。
- 本章阐述了开头呈示部（exposition）段落的一些关键原则。
- 我们引入了新术语，包括以下（点击术语查看词条）：主题（Subject）答题（Answer）对题（Countersubject）呈示部（Exposition）
- 我们还遇到了一些赋格特有的常用术语变体，特别是：声部（Voices）SATB

我们现在从十六世纪常见的模仿实践转向赋格，可以将其视为这一传统的十八世纪对应和继承者。赋格可能有很大差异，因此"赋格"是一个很难令人满意地定义的术语。赋格通常是复调作品，由严格使用一定数量的独立声部（voices）来定义；每个声部逐一进入，陈述赋格的主题（subject）。在本章中，我们将赋格视为一个过程，并展示如何从基本原理构建（第一部分的）赋格。本章涉及根据巴赫（Bach）、亨德尔（Handel）及其同时代人的"高巴洛克"技法，从给定主题写作赋格呈示部的基础知识。

# 基本定义

首先，让我们定义一些我们将需要的术语。

- 主题（Subject）：构成赋格旋律基础的短曲调，贯穿全曲反复出现。
- 答题（Answer）：主题的移位，有时也稍作改变，如下所述。
- 对题（Countersubject）：与主题/答题一起发声并对其进行补充的旋律线条。
- 呈示部（Exposition）：赋格的第一部分，在此期间每个声部依次进入陈述主题或答题。注意这与奏鸣曲式（sonata form）中的"呈示部"用法不同；两者都指设置主要材料的第一段，但在其他方面有所不同。
- 声部（Voices）：独立的对位线条，可以是器乐的或声乐的。声部数量不一定与部分数量或演奏者数量对应——一个演奏者的键盘赋格可能有三、四、五或六个声部，甚至有些独奏小提琴赋格（这相当是作曲上的壮举！）。
- SATB：虽然赋格可能不涉及任何歌手，但习惯上用 S（高音/soprano）、A（中音/alto）、T（次中音/tenor）和 B（低音/bass）来指称四声部织体中的线条，从高到低排列。音域不必与声乐部分完全匹配。

# 结构 / 声部进入

基本上，赋格呈示部如例 1 所示运作：一个声部先开始陈述主题，然后下一个声部进入陈述答题，而第一个声部继续进行对题，依此类推。

声部 1 | 主题 | 对题 | 自由对位
声部 2 | | 答题 | 对题
声部 3 | | | 主题

例 1. 赋格开头声部进入的假设结构。

注意声部按照进入顺序编号。虽然声部按从高到低或从低到高的顺序进入非常普遍，但这并非严格必要。然而，当一个新声部进入时，它应该是该阶段的最高声部或最低声部，因此中间声部（中音或次中音）最后进入是不常见的。这给了我们足够的选择。例如，四声部赋格的声部进入顺序可以是：

- 按音域从高到低（SATB），或反之（BTAS）
- 从中音开始：ASTB, ATSB, ATBS
- 从次中音开始：TBAS, TABS, TASB

任何超出声部总数的主题/答题进一步进入被描述为"冗余"进入。

# 主题

在考虑如何处理主题时，要观察其结构和特征。

主题的结构通常可以分为三部分。像许多调性乐句一样，主题描绘了从特征性开头到一般性终止的路径：

- 头部动机（Kopfmotiv，字面意思为"头部动机"）：开头的手势
- 展开（Fortspinnung）：可能包括模进（sequence）和/或持续律动（motor rhythm）的延续
- 终止（Cadence）

特征类型包括：

- 托卡塔风格（Toccata style）：具有持续律动的感觉，经常是半音化的，可能有不规则的节奏安排。
- 利切卡尔风格（Ricercar）：源自圣歌/经文歌传统，速度缓慢，具有古风感，并大量使用延留音（suspensions）。

# 答题

我们区分两种类型的答题：完全移位的真实答题（real）和进一步改变的调性答题（tonal）。

真实答题简单地将原始主题移位纯五度/四度。音符之间的所有一般音程关系保持不变。

我们需要调性答题，如果主题：

- 从 sol $(\hat5)$ 开始或在开头显著使用它。在这种情况下，我们必须调整移位后的版本，使 sol $(\hat5)$ 不被向上移位为 re $(\hat2)$，而是到 do $(\hat1)$。
- 在开头包含任何其他 V-I 的暗示。主题中突出的属音在答题中变成主音，因此 T → D 的进行通常变成 D → T，反之亦然。
- 转调。在这种情况下，将主题分成两个乐句，每个在一个调中，并相应调整。如果主题在结尾处转向属调，答题必须转回主调。

在所有情况下，我们寻求进行最小调整，使主题和答题仍然尽可能相似，且两者在旋律上都有意义。

# 对题

在完全展开之前，先试着构建对题的骨架版本。匆匆写出一段优美的曲调然后发现它不合适，这很令人沮丧。此外，在勾勒与主题搭配的版本时，同时勾勒与答题搭配的版本。写出一个与主题完美配合但答题使和声上毫无意义的对题同样令人沮丧。

确实，对题是旋律，但只有在和声上也有意义时才有效。为此，请记住你的对题将首先出现在两声部段落中，因此考虑如何用仅两个声部勾勒出清晰的较大型和声（三和弦和七和弦）。

你可能希望给自己设定额外约束，即对题可在八度上倒影——基本上是一个可以出现在主题上方或下方且仍然有效的对题。在这种情况下，同样要同时处理两个版本，特别确保：

- 避免五度，因为五度倒影后变成四度，在两声部写作中有特殊规则。坚持使用同度（/八度）、三度（/六度），在七和弦的情况下也包括二度（/七度）。五度在自由声部中不是问题，因此将该音符"保留"作为和弦的完成并在那里使用。
- 同样，避免 4-3 延留音，因为第一个音符在倒影后不是不协和的（5-6）。相反，使用 7-6 和 2-3 延留音（它们互相倒影）。

要创建骨架对题，首先在主题中寻找骨架模式。找到主题的简单、无装饰形式，将其视为一种调性固定旋律（cantus firmus），在其上写作你的对位（骨架对题），然后可以将其装饰为更有趣的音乐线条。

在装饰方面，试着：

- 在创作对题时忠实于主题的特征
- 建立互补的节奏运动，使一个声部运动时另一个声部静止
- 仅在主题/答题中发生变化的地方对对题进行相应改变

# 自由对位

坦白说，"自由"对位的概念有点用词不当，特别是如果你在写作可倒影的对题，因为主题和对题将使你的选择受到高度限制。这部分的"自由"在于它不一定在后面再次出现，所以你可以把它看作一个更自由的部分，相对于更不自由的主题和对题而言！或者，你可以拥抱极端的秩序，用一个后面确实再次出现的正规对题来替代自由对位。

# 连接

你的赋格呈示部可能包含主题/答题进入之间的短小插入。我们称这些为连接（links）。连接经常使用主题中的动机，例如对短片段的模进重复。它们可以出现在任何或所有声部进入之间，长度也可以不同。

包含一个或多个连接有许多音乐上的动机。连接提供了以下机会：

- 改变和声：一个主题/答题的结束和下一个的开始可能在和声上不匹配。在这种情况下，你可以使用连接到达你需要的地方。
- 改变音域：如果你有一个不断下降的线条，例如，连接可以是重置音域的有用方式，使你不超出乐器的范围。
- 变化乐句长度：如果你有一个恰好两小节的主题，只是按顺序通过几个声部进入，那么你可能想使用连接使超节拍和乐句分组更有趣。话虽如此，声部始终应在节拍上可比较的位置进入：例如，如果你的主题在弱起拍上进入，所有主题/答题进入也应如此，至少在呈示部中是这样。
- 搭叠进入：除了变化乐句长度，我们还可以变化声部进入的旋律语境。例如，如果一个连接对头部动机进行模进发展，那么你可以无缝地引入下一个主题/答题，因为它从该动机开始。

# 一般注意事项

除了赋格的具体考虑之外，自然许多写调性音乐的一般实践在这里适用。特别记住要：

- 控制节奏流动：如果当某一节拍级别的持续使用已经积累起来（例如四分/八分/十六分音符），要仔细考虑是否以及何时停止该运动。
- 分级终止（cadences）的相对强度，例如通过控制最高和最低声部的音级。
- 为乐器写出地道的音乐。记住观察手跨度、乐器范围等方面的实际限制。
