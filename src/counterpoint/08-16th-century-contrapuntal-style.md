II. 对位法与华丽风格程式

16世纪对位风格
Mark Gotham
要点

- 16世纪对位风格与对位法程式中概述的原则相关——但并不相同。
- 本章设定了一些关键原则和练习，特别是关于模仿（imitation）的练习。

16世纪对位风格在音乐理论教学中历来占有重要地位。16世纪模仿合唱复调音乐的"仿作"或"反事实"作曲（特别是帕莱斯特里那风格）经常出现在课程中，有时与我们前面章节中遇到的较晚的18世纪对位法程式概念相混淆。本章设定了一些经验法则，在完成基于该曲目的风格作曲练习时（在这些练习中你会收到一份部分乐谱来完成）需要牢记。最后，你将找到几个这种格式的示例练习。

# 模仿

模仿涉及两个或多个声部分别进入，演唱相同的旋律或相同旋律的变体。这是16世纪对位音乐中的常见做法，特别是用于开始整个乐章和乐章中的大型段落（例如引入新的歌词行）。你可以将其视为后来赋格曲（fugue）的前身（我们将在《盛期巴洛克赋格呈示部》章节中再次讨论）。

以下是维森特·卢西塔诺（Vicente Lusitano）的"Ave spes nostra"的示例，出自

_Liber primus epigramatum_

在这个开头段落中，一个设置"Ave spes nostra"歌词的"模仿点"出现在所有五个声部中。模仿是精确的，只有微小的变化。第一个进入的声部从F开始下行到C，然后它第二次重复歌词，现在从C开始下行到F，使用类似的旋律，在最后多加一个音符。其他声部模仿这个模式（F–C然后C–F）或反过来（C–F然后F–C）。

这个示例的末尾看到了第二个模仿点的开始，设置下一行歌词"Dei genitrix..."。这个新图形具有"回音"般的形态：原音、上方音、下方音、原音（C-D-Bb-C）。接下来是基于第二个点的六声部段落。在这种风格中很常见，两个乐句重叠（即新乐句在前一个乐句结束之前开始），第二个点的模仿比第一个处理得更自由。你甚至可以在这个最后小节中看到第一个点的几个微妙的节奏变化，这正是重叠发生的地方。

当完成涉及模仿的练习时，让现有材料作为你的指南，并考虑以下指导原则：

- 识别（潜在的）模仿点：在任何给定的声部中，寻找歌词的变化（通常前面有逗号或休止符），特别是任何用相同音乐重复歌词的地方。
- 音高音程：段落或乐章开始时模仿声部之间的音程通常是完全协和音程。也就是说，其他的也完全可能，特别是在乐章的较后部分。请注意，我们谈论的是声部之间的主要对应音程，而不一定是第一个音。我们有时会看到后面盛期巴洛克赋格中典型的调性应答（tonal answer）的例子，在下一章中讨论。
- 节拍：在节拍可比的位置进行模仿（强拍→强拍；弱拍→弱拍）。
- 当研究哪些模仿关系可行时，尝试双向的所有选项。由于一个模仿点所管辖的段落通常与下一个重叠，围绕该段落变化的模仿进入可能来自另一个点。
- 确保连续进入的模仿声部之间有有意义的重叠。为此，较短的模仿点可能需要相应较短的声部进入间隔。
- 考虑模仿多少该点的内容。较长的旋律最终会从精确模仿的点变为自由对位。在这种情况下，这种变化发生在哪里？
- 考虑模仿该点的紧密程度。至少保留该点开头的节奏和独特的音程/轮廓/形态。
- 记住乐句和段落可以重叠，即使在强终止处也是如此。

# 旋律

- 反向运动（contrary motion）在声部之间占主导地位（部分是为了保持声部的独立性）。避免过多的平行写作，即使是允许的音程（但注意"福布尔东"（fauxbourdon）的例外情况，即一段延长的平行三和弦段落）。
- 级进运动在声部内部占主导地位。以级进接近终止音（在大多数声部中）。从下方接近终止音时升高导音（在大多数调式语境中，弗里吉亚调式除外）。
- 旋律音程：二度、三度、四度、五度和八度都是允许的；六度的状态更为复杂。上行小六度使用相当自由（在帕莱斯特里那弥撒曲中有814次），上行大六度较罕见（54次），下行六度极其罕见（7次下行小六度，仅1次下行大六度）。七度和三全音（tritone）是被避免的。甚至不要用连续跳进或旋律手势的边界（高点和低点）来勾勒这些音程。通常避免同向连续跳进。如果有这样的连续跳进，将较大的跳进放在较低的位置（上行时在前面，下行时在后面）。大跳进之后（通常之前也）应接反向运动。这与填补空缺（gap-fill）和回归音域中间的概念有关。避免以分解和弦的方式勾勒三和弦。
- 轮廓：拱形是旋律的常见形态（上去的终要下来！）。
- 每个声部的音域：一个常见的建议是将每个声部保持在八度以内，对应于作品调式的正格（authentic）或副格（plagal）音域。但在实践中，声部通常跨越略宽的14个半音范围。极少有声部超过十二度（八度加五度），因此将其作为最大值。
- 常见的旋律手法：延留音：延留音符应在一个弱拍位置上作为协和音"准备"，连线到强拍位置上的不协和音，并通过将延留的不协和声部下行一个级进来解决。对于7-6和4-3延留音，这涉及上声部的不协和；对于2-3延留音，不协和在下声部（作为7-6的转位）。这在第四程式对位法章节中有进一步解释。虽然不是延留音，但从五度到六度的斜向运动也很常见。乐句末尾的装饰比开头更常见。

# 节奏与节拍

- 该曲目的原始记谱没有小节线，但节拍思维非常清晰。虽然现代版本通常会明确标注小节线，请记住这些是编辑干预，而非原始的。编辑也可能将tactus级别（半拍级别）的音符时值改为比原始记谱更短的音符（例如从二分音符改为四分音符）。在牢记这些注意事项的前提下，编辑干预可以是有帮助的——小节线有助于排练和指挥，一些演奏者可能觉得用更短的音符时值读谱更舒服。编辑选择也可能提供更微妙的提示，例如使用"更长"的节拍（用二二拍代替四四拍）来暗示以更长拍子思考的可能性，从而暗示更长不协和音程的可能性。
- 旋律线通常以较慢的节奏运动开始（且通常结束）。
- 半tactus（四分音符）运动应从弱拍位置开始——在非重音拍（第二或第四拍）上，或在拍子之间（作为附点节奏的一部分）。
- 连线（ties）将长音连接到短音（而非相反）。
- 在（相对罕见的）三拍子情况下，节奏通常分为2+1而非1+2（与许多风格一样）。

# 歌词配置

- 许多这个时期的作曲家高度重视音乐和歌词的清晰度。[2]
- 节拍：尽管有上述注意事项，通过将强音节放在"强拍"上来匹配歌词和音乐的节拍。系统性例外遵循上述惯例，包括切分、延留音和"节拍不协和"（在一个声部中使用与其它声部中占主导地位的节拍相反的一致节拍——通常是现代版本中记谱的节拍）。
- 音乐乐句跟随歌词。通常每个音节至少使用一拍（二分音符）。例外包括附点节奏，其中较短的（四分）音符经常接受自己的音节。这可以被视为对"直"节奏的修改，符合"每拍一个"的指导原则。在半tactus（四分音符）运动之后立即更换音节是罕见的。

# 织体

- 适当的织体通常从练习中给出的部分来看是清楚的。
- 粗略地概括，弥撒曲设置中织体的广泛惯例如下（I = 通常为模仿；H = 可能为[更]主调）：垂怜经（I）。荣耀经（H）。"qui tollis"经常作为独立段落存在。信经（H）。在重要时刻尤其常为主调。"Crucifixus"可分离。圣哉经（I）。三呼圣哉（H）。降福经（I）。通常用于较少声部。（第二个三呼圣哉通常是第一个的重复。）羔羊经（I）。可能有第二个或第三个羔羊，通常有更多声部和卡农。
- 圣母颂歌（Magnificat）倾向于是一个更自由的体裁，对模仿点的处理更加灵活。

# 和声

这个时代的作曲家不像我们今天通常所做的那样以和弦、罗马数字、转位等方式思考。相反，他们主要关注声部之间的音程关系。考虑到这一点，以下是关于这种风格中声部纵向组合的一些指导原则。

- 平行五度和平行八度：避免，与后来的惯例一样。然而，与后来的惯例不同，涉及减五度的也必须避免。
- 直接五度或八度：原则上避免（一些人以它们暗示了如果添加中间经过音就会出现的平行进行来合理化这一点）。在终止处和/或当被一个或多个其他声部的强反向运动缓解时更为常见。
- 减三和弦：在帕莱斯特里那的音乐中使用并不罕见，主要为第一转位。特别是在多声部音乐中作为带有低音上主音到主音运动的终止式的解决方案。按照标准声部进行解决：re-do、ti-do、fa-mi。
- 不协和音：类型和处理：延留音：准备（弱拍）– 延留（强拍）– 解决（弱拍）。"非本质"不协和音：半tactus（四分音符）级别的经过音、邻音。变化音，"变化音"：通常为do-ti-sol-la。唯一从不协和音跳开的情况。
- 最终和弦：空五度或大三和弦（皮卡第三度）。升高（"皮卡第"）三度"起源于约1500年，当时首次在作品的最终和弦中接纳了三度……在16世纪下半叶，这种做法变得相当普遍"（Apel 1969, 677）。在帕莱斯特里那时，大三和弦是最常见的最终和弦类型（在帕莱斯特里那弥撒曲中约占85%的案例）。
