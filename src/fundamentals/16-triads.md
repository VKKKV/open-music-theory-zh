I. 基础

三和弦
Chelsey Hamm

要点

- 三和弦（triad）是一个三音和弦，其音符可以按三度排列。三和弦总是可以"叠加"，使其音符全在线上或全在间上。
- 当三和弦以最紧凑的三度形式叠加时，最低的音符称为根音（root），中间的音符称为三音（third），最高的音符称为五音（fifth）。
- 三和弦有四种性质。大三和弦（major triad）的三音是大三度，五音是纯五度；小三和弦（minor triad）的三音是小三度，五音是纯五度。减三和弦（diminished triad）的三音是小三度，五音是减五度；增三和弦（augmented triad）的三音是大三度，五音是增五度。
- 在和弦符号（chord symbols）中，大三和弦用对应根音的大写字母表示。小三和弦在字母后加小写的"mi"，减三和弦用上标圆圈（o），增三和弦用加号（+）。你可能会在其他语境中看到其他缩写（更详尽的列表见和弦符号章节）。
- 在大调和小调中，三和弦具有与音级对应的特定性质。这些在每个大调和小调中都是相同的，因此记住它们很有用。
- 三和弦根据其根音、性质和转位来识别。

和弦（chord）是三个或更多音级同时发声的任何组合。本章重点讨论三和弦——即音符可以叠加为三度的三音和弦。

# 三和弦

三和弦的三个音符总是可以按三度排列。例1展示了两个三和弦，每个都以旋律性和和声性两种方式书写。第一个三和弦在三个相邻的间上，第二个三和弦在三个相邻的线上。三和弦总是可以"叠加"，使其音符全在线上或全在间上。当三和弦以最紧凑的形式叠加时（例1的第2和第4小节），它看起来像一个雪人（见例2）。正如雪人由底部、中部和头部组成，三和弦由最低、中间和最高的音符组成。

通俗地说，当三和弦以这种方式叠加时，我们可能称之为"雪人形式"，但正式术语是原位（root position）；这在名为转位的章节中有更多探讨。如例3所示，三和弦最低的音符称为根音，中间的音符称为三音（根音上方的通用三度），最高的音符称为五音（根音上方的通用五度）。

# 三和弦性质与聆听三和弦

三和弦有四种性质——大、小、减和增——由根音到三音和根音到五音的音程性质决定。这些性质在例4中标注：三和弦性质显示在五线谱上方，五音到根音的性质在三和弦左侧用红色标出，三音的性质在三和弦右侧用蓝色标出。增三和弦放在最后展示，因为其他三种类型在大多数古典和流行音乐中更为常见。

请注意，三和弦根据它们包含的某些重要音程来命名：

- 大三和弦和小三和弦根据其三音的性质命名。（两者都有纯五度。）
- 减三和弦和增三和弦根据其五音的性质命名：减三和弦的五音是减五度。（其三音是小三度。）增三和弦的五音是增五度。（其三音是大三度。）

仔细聆听例4中不同性质的三和弦。在学习三和弦的音色时，将表现性质与三和弦配对是常见的做法。你可能会觉得大三和弦听起来"快乐"，小三和弦"悲伤"，减三和弦"可怕"，增三和弦具有"幻想"或"神秘"的音色。

例4. 不同性质的三和弦。

# 和弦符号

三和弦的和弦符号包括根音的字母名称和三和弦性质的指示，有时还包括低音声部的音级（即和弦中最低的音符，而不是任何特定的乐器或声部类型）。

和弦符号以大写字母（必要时加上变音记号）开始，表示和弦的根音。该字母后跟关于和弦性质的信息。Open Music Theory将使用以下缩写。[1]

- 大三和弦：不加性质符号
- 小三和弦：小写"mi"
- 减三和弦：上标圆圈（o）
- 增三和弦：加号（+）

例如，和弦符号C、Cmi、Co和C+分别表示C大三和弦、C小三和弦、C减三和弦和C增三和弦。如果和弦的根音有变音记号，请包含它：例如，B♭mi是B♭小三和弦的和弦符号，F♯o是F♯减三和弦的和弦符号。

最后，如果和弦根音以外的音级是和弦中的最低音，则添加斜线，后跟表示低音（bass）音级的大写字母。这个主题将在名为转位的章节中进一步探讨。例5首先展示了C大三和弦和C小三和弦及其和弦符号（C和Cmi）。然后展示了当C以外的音符在底部时，C小三和弦的和弦符号如何变化：当E♭是最低音时，和弦符号为Cmi/E♭，当G是最低音时，符号为Cmi/G。

例5. 展示了四个三和弦及和弦符号。

# 大调和小调中的三和弦性质

大调音阶的任何音符都可以成为三和弦的根音。如例6所示（在G大调中），在大调中建立在do、fa和sol$(\hat1,\ \hat4,$ 和 $\hat5)$上的三和弦是大三和弦，用根音的大写字母表示。建立在re、mi和la$(\hat2,\ \hat3,$ 和 $\hat6)$上的三和弦是小三和弦，在根音大写字母后加小写"mi"。建立在ti$(\hat7)$上的三和弦是减三和弦；用上标"o"表示（你可能知道这是度数符号）。这些三和弦性质在不同的调中不会改变；换句话说，在任何大调中，建立在do$(\hat1)$上的三和弦性质始终是大的。

例6. 大调中的三和弦性质。

例7在G小调中，展示了小调音阶的三和弦性质模式。请注意，这个例子包含两个建立在sol$(\hat5)$上的三和弦和两个建立在te/ti$(\hat7/\uparrow\hat7)$上的三和弦——一个没有升高导音（"自然小调"），一个有升高导音（"和声小调"）。如例7所示，建立在do、fa和sol$(\hat1,\ \hat4,$ 和 $\hat5)$（没有升高导音）上的三和弦是小三和弦（用小写"mi"表示）。建立在me、le和te$(\hat3,\hat6,$ 和 $\hat7)$（没有升高导音）上的三和弦是大三和弦。建立在sol$(\hat5)$上带有升高导音的三和弦也是大三和弦。建立在re和ti$(\hat2$ 和 $\uparrow\hat7)$（有升高导音）上的三和弦是减三和弦（用上标度数符号表示）。

# 拼写三和弦

要从和弦符号构建三和弦，你需要了解三和弦的根音和性质。完成以下步骤：

- 在五线谱上画出根音。
- 画出根音上方三度和五度的音符（即画一个雪人）。
- 想出（或写下）三和弦根音的大调调号。
- 要拼写大三和弦，写下适用于三和弦音符的调号中的任何变音记号。
- 对于小、减或增三和弦，在适当时添加额外的变音记号来改变和弦的三音和/或五音。

例8展示了D大三和弦的过程：

例8.
用四个步骤画一个D大三和弦。

- 音符D，和弦的根音，被画在五线谱上。
- 画了一个雪人——F和A，D上方的通用三度和五度的音符。
- 回忆了D大调的调号。D大调有两个升号，F♯和C♯。
- 在F的左边添加了升号（♯），因为F♯在D大调的调号中。不需要C♯，因为和弦中没有C。

让我们对A♭小三和弦（A♭mi）完成这个过程，如例9所示。

例9.
用五个步骤画一个A♭小三和弦。

- 写下音符A♭，因为它是三和弦的根音。
- 画了一个雪人；换句话说，添加了音符C和E，因为它们分别是A♭上方的通用三度和五度。
- 回忆了A♭大调的调号。A♭大调有四个降号：B♭、E♭、A♭和D♭。
- 添加了E♭，因为它在A♭大调的调号中。不需要B♭和D♭，因为这些音符不在A♭三和弦中。现在我们已经成功拼写了A♭大三和弦（A♭、C和E♭）。
- 小三和弦包含小三度，比大三度小一个半音。因此，我们的最后一步是将和弦的三音（C）降低一个半音（到C♭）。现在我们有了A♭小三和弦（A♭、C♭和E♭）。

不要忘记减三和弦有小三度和减五度，意味着你必须将三音和五音都从大三和弦降低一个半音。增三和弦有大三度和增五度，所以其五音必须从大三和弦升高一个半音。

# 识别三和弦、重复音和间距

三和弦根据其根音、性质和转位来识别；转位在转位章节中讨论，所以我们现在将专注于根音和性质。你可以通过三个步骤识别三和弦：

- 识别并写出其根音。
- 想象其根音的大调调号。
- 识别并写出其性质。

例10展示了一个用于识别过程的三和弦。

例10. 一个用于识别的三和弦。

要识别这个三和弦：

- 这个三和弦以最紧凑的形式书写，所以根音是最低的音符，D。
- D大调有两个升号，F♯和C♯；我们可以利用这个信息来识别三和弦的性质。
- 在这个三和弦中，F是升的，与大调调号匹配。因此，我们可以正确地将其识别为D大三和弦。

例11. 第二个用于识别的三和弦。

要识别这个三和弦：

- 首先，我们识别并写出其根音，即C♯。
- 接下来，我们可以识别并写出其性质。我们想象C♯大调的调号，它有七个升号（每个音符都是升的）。
- 因此，在C♯大调中E和G应该是升的，但我们在例11中看到它们都是还原的。因为三音和五音都降低了半音，这个三和弦是减的。我们现在可以正确地将这个三和弦识别为C♯o。

如果三和弦的底部音符有想象的调号（因为有一个适用于它的双变音记号），请使用等音等价来重新拼写三和弦。这个过程将与音程章节最后一节中概述的相同。
