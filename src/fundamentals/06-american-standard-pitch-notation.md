# 美国标准音高标记法与音高对音级

为了讨论特定的音符或音高（pitch），我们将使用美国标准音高标记法，缩写为 ASPN。ASPN 通过将音符名称（如 C）与下标八度（octave）标记（如 4）相结合来指定特定的音乐音高，创建一个两部分的标签（例如 C4）。ASPN 标签非常有用，因为它们可以识别人类听力范围内所有可能的音乐音符，从最低音到最高音。

阅读谱号一章介绍了八度等价（octave equivalence），即解释为什么相隔一个或多个八度的音符具有相同字母名称的概念。音乐理论家区分音高（pitch）和音级（pitch class），前者是具有独立频率的离散音调（如使用 ASPN 的 C4），后者则不太具体（如泛指的 C）。一个音级包括所有具有相同字母名称的音高，在任何八度中，以及它们的等音等价音。例如，所有的 C 都是同一个音级，而等音等价的音符重降 D 和升 B 也属于 C 音级；更多信息请参见音高与音级。在本章中，我们使用 ASPN 来命名特定音高。

# ASPN 与八度标记

ASPN 区分八度（octave），从音高 C 开始，到音高 B 结束。[1] 这意味着每个新的八度标记都从音符 C 开始，如示例 1 所示。八度从最低到最高标记，从 0 开始，按数字顺序排列（1、2、3 等）。音高中央 C 是 C4，记住这一点很有用。

示例 1.

八度内的所有字母名称（下一个八度的 C 以下）都获得相同的八度标记。例如，示例 2 中的所有音符都将被指定为第 4 八度，因为它们在 C4 之上但在 C5 之下。应用于音符的变音记号不会影响其 ASPN 数字。例如，升 B3 和 C4 具有不同的八度数字，尽管它们是等音等价的，因为升 B 仍被认为属于较低的八度。

# ASPN 与键盘

ASPN 标签对于在钢琴键盘上找到特定音符非常有帮助。示例 3 描绘了一个用 ASPN 标记标注每个八度的钢琴键盘。如你所见，钢琴键盘跨越了完整的 1 至 7 八度。它还包含八度 0 和 8 的一小部分。ASPN 标签与乐器或声部类型无关。换句话说，C4 无论由长笛、长号、小提琴还是人声产生，都将始终被标记为 C4。

- 你可能想知道为什么八度数字随音高 C 而变化，而不是随 A（字母表的开头）变化。关于这一点，Robert Young 写道："出于音乐目的，已形成将音调 C 视为计算完整八度的起点的惯例。虽然这并非普遍适用，逻辑上可能指向 A 作为起点，但通常用法已确定为 C。" 见 Robert W. Young, "Terminology for Logarithmic Frequency Units," Journal of the Acoustical Society of America 11 (1939): 135.↵
