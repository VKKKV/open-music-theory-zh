III. 曲式

混合乐句级形式（Hybrid Phrase-Level Forms） John Peterson

关键要点

- 混合形式（hybrid form）将一个原型的开始与另一个原型的结尾结合在一起。可能的开始是前句（antecedent）、呈示部（presentation）或复合基本乐思（compound basic idea）。可能的结尾是后句（consequent）、延续部（continuation）或终止性结尾（cadential ending）。
- 尽管任何开始都可以与任何结尾结合，但某些配对比其他更常见（见例7）。

# 什么是混合形式？

在《乐句、原型与独特形式》中，我们看了两种乐句级形式原型：乐句（sentence）和乐段（period）。

我们发现这些形式分为两部分。乐句分为两个分句（subphrases）：呈示部（presentation）和延续部（continuation）。乐段分为两个乐句（phrases）：前句（antecedent）和后句（consequent）。

例1。贝多芬《钢琴奏鸣曲 Op. 13》第二乐章，第1-8小节中的前句+延续部（[0:00–0:32](https://open.spotify.com/track/2zSiV8Ze0yyrFtbVzX2Ytg?si=-Zj8tWQQSYSAf1mSPCv-xw)）。

混合形式（hybrid form）出现在一个原型的开始与另一个原型的结尾配对时。在例1中，前句之后是延续部。

- 我们如何知道第一部分是前句而不是呈示部？它以较弱的终止结束，而呈示部不以终止结束。它有一个基本乐思（basic idea）后接对比乐思（contrasting idea），而呈示部有一个基本乐思后接基本乐思的重复。
- 我们如何知道第二部分是延续部而不是后句？它不像后句那样以基本乐思开始，而是以片段化（fragmentation）开始，这是延续部的特征。它以半终止后持续的属音所创造的不稳定感开始，而不是后句开始时典型的稳定性。

到目前为止，我们知道两种开始（呈示部和前句）和两种结尾（延续部和后句），但实际上还有另一种可能的开始（复合基本乐思）和另一种可能的结尾（终止性）。下面，我们将概述这些新的开始和结尾，并提供帮助区分它们的方法。

# 开始类型

## 新增：复合基本乐思（c.b.i.）

例2。海顿《C大调钢琴奏鸣曲》Hob. XVI:35, I，第1-8小节中的复合基本乐思（[0:00–0:05](https://open.spotify.com/track/2MX7ZGLIM4lTBP8rIEbnXU?si=00V9I8XNTGC-7urHU2ovLQ)）。

复合基本乐思（compound basic idea, c.b.i.）是没有终止的前句，如例2所示。[1]

- 例2如何像前句？它包含一个基本乐思后接对比乐思。
- 例2与前句有何不同？它不以终止结束。注意低音在第3-4小节保持在G上：虽然旋律到达了休息点，但低音中缺乏和声运动规避了可能出现在那里的半终止（HC）。

## 摘要：三种开始类型

出现在混合乐句级形式中的三种开始是前句、复合基本乐思和呈示部。例3提供了区分每种开始特征的总结。

| 开始         | 以终止结束？ | 第一乐思 | 第二乐思 |
| ------------ | ------------ | -------- | -------- |
| 前句         | 是           | b.i.     | c.i.     |
| 复合基本乐思 | 否           | b.i.     | c.i.     |
| 呈示部       | 否           | b.i.     | b.i.     |

# 结尾类型

## 新增：终止性（cad.）

例4。海顿《弦乐四重奏 Op. 64, No. 4》第二乐章，第1-8小节中的终止性结尾（[0:05–0:10](https://open.spotify.com/track/2Dyh8LMXHEv4Hxchepyubj?si=1IMP3wmEQ_ut75jSngeJsA)）。

终止性（cadential, cad.）结尾为一个特定的低音模式配和声（在例4中演示）。[2]

核心低音模式是 mi–fa–sol–do$(\hat3-\hat4-\hat5-\hat1)$。[3] 这些音符的常见和声配置是：

- mi$(\hat3)$：I6
- fa$(\hat4)$：ii6 或 IV
- sol$(\hat5)$：V(7)（常用终止 $^6_4$ 装饰）
- do$(\hat1)$：I

这个核心 mi–fa–sol–do$(\hat3-\hat4-\hat5-\hat1)$ 低音线可以被装饰。一个常见的装饰是通过在 sol 之前添加 fi 来主音化属音（$(\uparrow\hat4-\hat5)$），使低音线变为 mi–fa–fi–sol–do$(\hat3-\hat4-\uparrow\hat4-\hat5-\hat1)$。

最清晰的终止性结尾为四小节长，每小节一个低音。

## 摘要：三种结尾类型

出现在混合乐句级形式中的三种结尾是后句、延续部和终止性。例5提供了区分每种结尾特征的总结。

| 结尾   | 终止           | 低音                                              | 乐思                             |
| ------ | -------------- | ------------------------------------------------- | -------------------------------- |
| 后句   | PAC/IAC，非 HC | 多样                                              | b.i. + c.i.                      |
| 延续部 | HC、IAC 或 PAC | 多样                                              | 多样，但常分为 1+1+2 或 2+2      |
| 终止性 | PAC/IAC，非 HC | Mi−fa−sol−do $(\hat3−\hat4−\hat5−\hat1)$ 或其展开 | 2+2 或不可分割。从不出现片段化。 |

- 复合基本乐思有什么"复合"的？它通常作为大型乐句的基本乐思，该乐句为十六小节长，其中呈示部为八小节长并包含两个 c.b.i. 单位。它之所以"复合"，是因为这些大型基本乐思本身由两个单位组成：一个 b.i. 和一个 c.i.↵
- 终止性乐思和终止性结尾有什么区别？终止性乐思出现在延续部的结尾，实质上，终止性结尾就像终止性乐思被扩展为四小节并替代了延续部。这意味着"终止性"可以出现在两个形式级别：乐思级别和分句级别。↵
- 当然，这也可以出现在小调中，如 me–fa–sol–do$(\downarrow\hat3-\hat4-\hat5-\hat1)$。↵
