# Open Music Theory (OMT2) 中文翻译

开源在线音乐理论教材中文版，基于 [Open Music Theory (OMT2)](https://viva.pressbooks.pub/openmusictheory/) 翻译。

## 构建

需要 [mdBook](https://github.com/rust-lang/mdBook)：

```bash
cargo install mdbook
mdbook build    # 输出到 book/
mdbook serve    # 本地预览 http://localhost:3000
```
